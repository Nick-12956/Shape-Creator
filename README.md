# Shape-Creator
Creates different shapes as per users intrest, also prints shapes as per users size input.  
Made using C language and has over 333 lines of code.  

FEATURES  --->  
1. Different shape options (14 different Shapes)
2. Size options (starting from 1 with exceptions for some shapes)
3. Display using stars (some shapes have additional symbols)
4. 14 Different Shapes
